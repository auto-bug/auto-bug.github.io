Kleks Keyboard¶

36 keys should be enough for everyone

Ever since I've build the {ref}`project-175712` I have been very happy with it. I use it everyday for work and

leisure, and it has improved my typing habits and lessened wrist pains. Encouraged by this, I also made some tries to use a two-part ergonomic split keyboards, such as Chocolad Keyboard Hacking — but somehow I feel that I really prefer single-piece keyboards — they are easier to place on the desk.

So now I want to make a keyboard that combines the advantages of the hardcore ergonomic designs, with my own preferences.
