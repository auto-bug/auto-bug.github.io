Camera Shield for S2 Mini¶

OV2640 shield for the S2 Mini ESP32-S2 board.

With CircuitPython now supporting a camera, it would be nice to have a

camera shield to use with my robot project-180025 . The OV2640 camera module is pretty powerful (it can create JPEG data to stream), and the S2 Mini has enough pins to drive it, so the only inconvenience left are three voltages need for powering the camera. The shield takes care of that.
