Bigbug¶

a 200% version of Fluffbug robot

The parts available to us sometimes drive what we build. When I found

some 2g sub-micro servos in my drawer, I decided that my Fluffbug robot needs a 50% version, and so Wee Bug was created. Now I found some beefy MG886R servos, so it’s time for a 200% (not quite) version.
